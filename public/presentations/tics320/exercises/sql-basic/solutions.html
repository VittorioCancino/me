Query 11

Show the film titles together with the categories they belong to.

SQL query

FROM film f

JOIN category c ON fc.category_id = c.category_id;

Solution 23

Query 23

Show the customer names and inventory ids for rentals that have already been returned.

SQL query

Query 27

Show the titles of films that belong to the Comedy or Family categories.

SQL query

FROM film f
JOIN film_category fc ON f.film_id = fc.film_id
JOIN category c ON fc.category_id = c.category_id
WHERE c.name IN ('Comedy', 'Family');
